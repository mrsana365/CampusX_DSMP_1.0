# CampusX_DSMP_1.0
CampusX Data Science Mentorship Program (DSMP) 1.0 by Youtube channel Campusx



*   **`01-Python-Programming/`**: Fundamentals, OOPs, Exceptions, and Decorators.
*   **`02-Data-Analysis-Libraries/`**: Extensive practice with NumPy, Pandas, and Matplotlib/Seaborn.
*   **`03-SQL-for-Data-Science/`**: Database fundamentals, DDL/DML commands, and complex queries.
*   **`04-Mathematics/`**: Linear Algebra, Statistics, Probability, and Calculus for ML.
*   **`05-Machine-Learning/`**: Algorithms from Linear Regression to Ensemble techniques (XGBoost, Random Forest).
*   **`06-Case-Studies/`**: End-to-end projects like "Indian Startup Funding Analysis" and "Laptop Price Predictor".
